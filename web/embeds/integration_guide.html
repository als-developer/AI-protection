Integration Guide

Complete guide to integrating BioShield with your application

📋 Prerequisites

  • BioShield API key from admin panel
  • Python 3.11+ or Node.js 18+ for SDK users
  • Network access to https://api.bioshield.secure-bank.internal

🐍 Python Integration

pip install bioshield-sdk

from bioshield import BioShieldClient

client = BioShieldClient(api_key="sk_live_xxx")
result = client.audit_voice(
    bank_cluster_token="your_bank",
    channel_identity="call_center",
    frequency_deltas=[0.12, 0.12, 0.11]
)
print(result.verdict)  # "VERIFIED_HUMAN_AUTHENTIC" or "CRITICAL_SUSPECTED_DEEPFAKE"

📘 JavaScript/Node.js Integration

npm install bioshield-sdk

const { BioShieldClient } = require('bioshield-sdk');
const client = new BioShieldClient('sk_live_xxx');
const result = await client.auditVoice({
    bankClusterToken: 'your_bank',
    channelIdentity: 'call_center',
    frequencyAmplitudeDeltas: [0.12, 0.12, 0.11]
});
console.log(result.evaluationVerdict);

☕ Java Integration

// Add dependency to pom.xml
<dependency>
    <groupId>com.bioshield</groupId>
    <artifactId>bioshield-sdk</artifactId>
    <version>1.0.0</version>
</dependency>

BioShieldClient client = new BioShieldClient("sk_live_xxx");
AuditResponse result = client.auditVoice(AuditRequest.builder()
    .bankClusterToken("your_bank")
    .channelIdentity("call_center")
    .frequencyAmplitudeDeltas(Arrays.asList(0.12, 0.12, 0.11))
    .build());
System.out.println(result.getEvaluationVerdict());
