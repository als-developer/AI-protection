Widget Installation Guide

Add BioShield security to any website in minutes

🚀 Quick Installation

Copy and paste this code just before the closing </body> tag:

<!-- BioShield Security Widget -->
<script src="https://widget.bioshield.secure-bank.internal/widget.js"></script>
<script>
    BioShieldWidget.init({
        apiKey: 'YOUR_API_KEY_HERE',
        position: 'bottom-right',
        theme: 'dark'
    });
</script>

⚙️ Configuration Options

apiKeyYour BioShield API key
positionbottom-right, bottom-left, top-right, top-left
themedark, light, auto
autoDetecttrue/false

🎨 Custom Styling

.bioshield-widget {
    --primary-color: #ef4444;
    --bg-color: #0f172a;
}
