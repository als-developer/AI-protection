📦 Installation

Add this code just before the closing </body> tag:

<!-- BioShield Security Widget -->
<script src="https://widget.bioshield.secure-bank.internal/widget.js"></script>
<script>
    BioShieldWidget.init({
        apiKey: 'YOUR_API_KEY',
        position: 'bottom-right',
        theme: 'dark'
    });
</script>

⚠️ Note: Replace YOUR_API_KEY with your actual API key from the admin panel

⚙️ Configuration Options

apiKey Your BioShield API key
position 'bottom-right', 'bottom-left', 'top-right', 'top-left'
theme 'dark', 'light', 'auto'
autoDetect true/false - Auto-analyze audio streams
