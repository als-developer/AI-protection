API Documentation

Sovereign Bio-Shield Ultimate - Deepfake Detection API

Base URL

https://api.bioshield.secure-bank.internal/v1

🔑 Authentication

All API requests require an API key in the header:

curl -X POST https://api.bioshield/v1/audit-voice \
  -H "X-BioShield-Token: YOUR_API_KEY" \
  -H "Content-Type: application/json"
POST /audit-voice

Analyze a voice stream for deepfake detection

Request Body

{
  "bank_cluster_token": "string",
  "channel_identity": "string",
  "frequency_amplitude_deltas": [float],
  "metadata": {}
}

Response

{
  "security_token": "string",
  "evaluation_verdict": "VERIFIED_HUMAN_AUTHENTIC | CRITICAL_SUSPECTED_DEEPFAKE",
  "fraud_risk_index": "string",
  "firewall_action": "ALLOW | TERMINATE_CALL"
}

Example

curl -X POST https://api.bioshield/v1/audit-voice \
  -H "X-BioShield-Token: sk_live_key" \
  -d '{
    "bank_cluster_token": "crdb_hq_main",
    "channel_identity": "sip_trunk_01",
    "frequency_amplitude_deltas": [0.12, 0.12, 0.11]
  }'
GET /health

Check system health

curl https://api.bioshield/v1/health

⚠️ Error Codes

401 Invalid or missing API key
429 Rate limit exceeded
400 Invalid request payload
500 Internal server error
